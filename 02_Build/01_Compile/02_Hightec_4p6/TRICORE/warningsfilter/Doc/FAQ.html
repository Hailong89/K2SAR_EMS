  • The script stops with the following message:
    Perl v5.8.0 required--this is only v5.0.3, stopped at WarnungsFilter.pl line 18.
    The script is designed to work with Perl 5.8.0 or greater. Be sure you are using the good one.
    You can know your Perl version like that:
    [perl program path]/perl -v

  • The scripts stops with the following message:
    I can't open the file XX.html for writing
    or
    I can't close the file XX.html
    This happens when the hard disk (or the network share) is full, or when you don't have any write right on the destination path. Check you authorizations or your quotas.
